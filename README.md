# ServiceNow User, Group, and Role Management

## 📌 Overview
This project demonstrates basic ServiceNow administration tasks, including **creating users, groups, and roles, and implementing Role-Based Access Control (RBAC).**

## 🎯 Objectives
- Create users in ServiceNow
- Create groups
- Create roles
- Assign roles to users through groups only
- Validate access using impersonation

## 🛠️ Tasks Performed
- Created user records with details like name, email, and department
- Created groups for different teams
- Assigned roles to groups instead of directly to users
- Added users to groups
- Tested user access using impersonation

## 🔑 Key Concepts
- Role-Based Access Control (RBAC)
- User Administration
- Group Management
- Access Control
- Least Privilege

## 🧰 Tools Used
- ServiceNow Personal Developer Instance (PDI)

## ✅ Outcome
Successfully implemented secure and scalable access control by assigning roles through groups instead of individual users.
